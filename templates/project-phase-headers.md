# Project Phase Announcement Templates

## Simple / Minimal
```
─── Phase {N} ───
```

## With Label
```
───────────◆───────────
  PHASE {N}: {TITLE}
───────────◆───────────
```

## Triple Dot
```
· · · Phase {N} · · ·
```

## Usage
- Simple: short updates, quick handoffs
- With Label: phase completions, major milestones
- Triple Dot: lightweight, informal phase markers
- `{N}` = phase number, `{TITLE}` = phase name in caps
